Practicum 2.0: An Interactive Tool for Practicing Intro CS Topics
Senior Integrative Exercise at Carleton College

Practicum is an interactive, web-based tool that teaches introductory CS topics. It is a code-tracing and visualization program that allows students to take a guided walk through problems to better understand how these concepts work in a practical setting.

I lead a team of students to add object-oriented programming problems to the practicum project, extending the system to parse and simulate code with classes and objects, and also to visualize those changing variables during execution.
